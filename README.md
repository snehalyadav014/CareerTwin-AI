# AI Career Intelligence & Placement Management Agent

## Problem Statement
Students often struggle to understand their placement readiness, identify skill gaps, and create personalized learning plans.

## Solution
An AI-powered agent that analyzes resumes, identifies skill gaps, generates learning roadmaps, and helps students become placement-ready.

## Features
- Resume Analysis
- Skill Gap Detection
- Placement Readiness Score
- Learning Roadmap
- Admin Dashboard
